# Productivity-Tracking-Device
Device to track where my daily time is invested. It also helps to develop new hobbies, enhance health and many more.

FEATURES:
1. Hydration Alarm every 2 hour (Even hour of the day)
2. Prompt every 30 mins to document the type of work (Screen stays for 5-10 seconds)
3. To-Do list display, and chance to check mark the tasks.
4. All data collected and edited through Website or App.
   NOTE : App connectivity active only when user selects the option to connect.
5. Basic features like, STOPWATCH, TIMER, SET ALARM.
6. Try to keep different Watch faces. Only 3-4 will be stored on watch, rest will be available on app whicb can be uploaded to watch by replacing one of the watch faces.

PROTOTYPING STAGE 
Components Used :
1. ESP32 Dev Module
2. 1.3 inch OLED display
3. Breadboard
4. Wires
5. Push button
